#LẬP TRÌNH PHÂN TÍCH DỮ LIỆU VỚI PYTHON
Lưu trữ các bài tập và dự án cá nhân cho môn học **Lập trình Phân tích dữ liệu với Python**. Nội dung tập trung vào việc sử dụng hệ sinh thái Python để xử lý, làm sạch và trực quan hóa dữ liệu.

## 📂 Cấu trúc thư mục

* `Pandas/`: Các bài tập về thao tác dữ liệu với DataFrame và Series.
* `Homework/`: Tổng hợp các bài tập về nhà theo từng tuần.

## 🛠️ Công nghệ sử dụng

* **Python 3**
* **Pandas**: Thư viện chính để cấu trúc và phân tích dữ liệu.
* **NumPy**: Hỗ trợ tính toán mảng và ma trận.
* **Matplotlib & Seaborn**: Thư viện dùng để trực quan hóa dữ liệu.
* **Jupyter Notebook**: Môi trường lập trình tương tác.
